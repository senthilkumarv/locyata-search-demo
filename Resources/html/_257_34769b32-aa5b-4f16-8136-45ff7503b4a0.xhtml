Task 2: Verify a Frame Relay Connection
In this task, you will verify the configuration by using show commands and by executing a ping to the core router.

Step4
Verify that your Frame Relay map table lists a path to the core router by using the show frame-relay map command. Your output should be similar to the following example:
RouterA#show frame-relay map
Serial0/0/0 (up): ip 10.140.1.1 dlci 100(0x64,0x1840), dynamic,
              broadcast, status defined, active                        
Step5
Ping the core router serial interface that is directly connected to your workgroup router. Use the address that is provided in the Job Aids table of this lab activity.
Step6
Ping the TFTP server, 10.1.1.1.
All pings should be successful.
Step7
Proceed to Task 3.
Activity Verification
You have completed this task when you attain this result:
  • You have used show commands and successfully executed a ping to the core router to verify the Frame Relay configuration on your workgroup router.
