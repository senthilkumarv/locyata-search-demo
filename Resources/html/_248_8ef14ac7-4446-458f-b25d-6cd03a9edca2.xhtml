Task 2: Configure IPv6 Addresses
In this task, you will enable IPv6 globally on your router and configure IPv6 addresses on interfaces S0/0/1 and Loopback 2.
Activity Procedure
Complete these steps:
Step1
Use the shutdown command to shut down the first serial interface (S0/0/0) of your workgroup router that connects to the core.
Step2
Enable the second serial interface (S0/0/1) of your workgroup router by using the no shutdown command.
Step3
Enable IPv6 on your workgroup router.
Step4
Assign the IPv6 address that you determined in Task 1 to the second serial interface (S0/0/1).
Step5
Create a loopback 2 interface and assign it the IPv6 address that is determined in Task 1.
Step6
Display the IPv6 interface information to verify that all the interfaces on your workgroup router are configured with the appropriate IPv6 address.
Your output should be similar to the following example:
