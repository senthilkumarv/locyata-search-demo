In this lab activity, the second serial interface (S0/0/1) will use the following IPv6 address format:
2001:0410:000x:10::/64 eui-64
In this case, x = your group number that is listed in the Job Aids table of this lab activity
For example, the IPv6 address of the second serial interface (S0/0/1) of router A would be 2001:0410:0001:10::/64 eui-64.
The x:10 is the subnet portion of your IPv6 address. For this lab, it is important that the S0/0/1 interfaces that are in the same group have the same subnet address.
The EUI-64 parameter forces the router to complete the low-order 64 bits of the address (the host portion) by using the interface data link layer address (MAC address).
You will be creating a Loopback 2 interface on your workgroup router in Task 2. The loopback 2 interface for each workgroup router will use the following IPv6 address format:
2001:0410:000x:y::/64 eui-64
In this case, x = your group number that is listed in the Job Aids table of this lab activity and y = your router number that is listed in the table
For example, the IPv6 address of the Loopback 2 interface of router A would be 2001:0410:0001:1::/64 eui-64.
The subnet portion of the IPv6 address is :y. It is important for this lab that the loopback 2 interface of each router is in a different subnet.
