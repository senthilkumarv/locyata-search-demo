Lab 9-1: Implementing EIGRP
Introduction
Complete this lab activity to practice what you learned in the related module.
Activity Objective
In this activity, you will determine routes from a workgroup to a core site with EIGRP. After completing this activity, you will be able to meet these objectives:
  • Configure EIGRP on the router
  • Configure MD5 authentication for EIGRP
  • Verify the correct operation and configuration of EIGRP routing by using show commands, and verify the correct operation and configuration of EIGRP MD5 authentication
  • Debug the EIGRP neighbor processes
