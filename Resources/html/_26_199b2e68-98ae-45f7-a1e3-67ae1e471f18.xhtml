Task 1: Obtain the Complete Current IP Addressing Information
In order to obtain the complete current IP address information on your PC, it is necessary to use the Windows ipconfig /all command. To access Windows commands, it is necessary to open a Command window.
Activity Procedure
Complete these steps:
Step1
From the Windows desktop, click start.
Step2
Choose run, and enter cmd in the Run window dialog box. Click OK to continue.
Step3
From the Command window prompt, enter ipconfig /all. The /all parameter is used to show the complete output.
Step4
You will see from your own output that some extra, useful information is now visible.
