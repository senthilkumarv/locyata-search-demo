Task 2: Connect to Your Remote Workgroup via VPN Tunnel
In this task, you will open a VPN connection to your remote workgroup and then log into your assigned workgroup router by using the terminal emulation application. Use the username and password netadmin. You will then increase the vty lines automatic idle timeout to 30 minutes for the duration of this lab on your workgroup router.
Activity Procedure
Complete these steps:
Step1
From your PC, open a VPN connection to your designated workgroup.
Step2
From your PC, use PuTTY to SSH to the IP address of your workgroup router, 10.x.x.3. Use the username and password netadmin during this activity.
Step3
Get to the enable EXEC prompt, and enter the command show sessions. Your output should be similar to the following example:
login as: netadmin 
 
**********  Warning   *************
Access to this device is restricted to authorized persons only!
Un-authorized access is prohibited. Violators will be prosecuted. 
 
**************************************************************^C
netadmin@10.10.10.3's password: 
 
RouterX#show sessions
% No connections open
RouterX#
Step4
Enter the command show users to see the current users connected to your workgroup router. Your output should be similar to the following example:
RouterX#sh users
  Line         User       Host(s)              Idle       Location
*322 vty 0     netadmin   idle                 00:00:00 10.10.10.134
  Interface    User               Mode         Idle     Peer Address
The user “netadmin” is associated with the address of your PC because of the SSH connection via the VPN tunnel that you made in Steps 1 and 2 of this task.
Step5
Enter the command conf t to get to the global configuration prompt.
Step6
Enter the command line vty 0 4 to get to the vty line configuration mode.
Step7
Enter the command exec-timeout 30 to extend the idle timer period on your router vty lines to 30 minutes.
Step8
Return to the EXEC prompt by entering the command end. Your output should be similar to the following example:
RouterX#conf t
Enter configuration commands, one per line.  End with CNTL/Z.
RouterX(config)#line vty 0 4
RouterX(config-line)#exec-timeout 30
RouterX(config-line)#end
RouterX#
Activity Verification
You have completed this task when you attain these results:
  • You have connected from your PC to your remote workgroup router by using PuTTY via a VPN tunnel.
  • You have increased the idle timeout of the router vty lines to 30 minutes.
  • You have used the show sessions command to verify that the router has no open sessions at this time.
  • You have used the show users command to identify that you are the only user currently connected to your router.
