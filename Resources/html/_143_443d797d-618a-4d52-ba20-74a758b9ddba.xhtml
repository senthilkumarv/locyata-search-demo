Lab 5-1: Connecting to the Internet
Introduction
Complete this lab activity to practice what you learned in the related module.
Activity Objective
In this activity, you will be able to configure your WAN Ethernet interface to use a DHCP-obtained IP address, and will provide PAT. After completing this activity, you will be able to meet these objectives:
  • Use Cisco SDM to configure the WAN Ethernet interface to use a DHCP-obtained IP address
  • Use Cisco SDM to configure the router to support PAT of the inside Ethernet interface through the WAN Ethernet interface
  • Use Cisco SDM to verify that the configuration matches the requirements of the lab
  • Use the CLI to test and observe that PAT is taking place through the WAN Ethernet interface
