Lab Preparation
This lab activity requires that your instructor make changes to the core configuration. Verify with your instructor that the core configuration has been updated.
In this lab activity preparation, you will disable trunking on the workgroup switch ports Fa0/11 and Fa0/12, and return the access port to which your workgroup router is connected, Fa0/2, back to its default VLAN.
Activity Procedure
Complete these steps on your workgroup switch:
Step1
From global configuration mode, enter interface range configuration mode for ports Fa0/11 and Fa0/12.
Step2
Configure both switch ports Fa0/11 and FA0/12 as access ports.
Use the command switchport mode access.
Step3
Enable both switch ports Fa0/11 and Fa0/12 with the command no shutdown.
Step4
Enter interface configuration mode for the port to which your workgroup router is connected, Fa0/2.
Step5
Change switch port Fa0/2 to be a member of VLAN 1.
Use the command switchport access vlan 1.
