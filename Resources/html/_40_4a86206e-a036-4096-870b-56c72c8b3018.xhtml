Task 7: Explore Context-Sensitive Help
In this task, you will use context-sensitive help in both user and privileged EXEC modes to locate commands and complete command syntax.
Activity Procedure
Complete these steps:
Step1
Connect to your workgroup switch by using the information from the previous exercise.
Step2
Enter the help command (?). At the user EXEC prompt, you should see a partial list of commands that are available. Your output should resemble the following example:
Exec commands:
  access-enable    Create a temporary Access-List entry
  clear            Reset functions
  connect          Open a terminal connection
..
..Text omitted
..
  set              Set system parameter (not config)
  show             Show running system information
  ssh              Open a secure shell client connection
  systat           Display information about terminal lines
  telnet           Open a telnet connection
 --More--
Step3
Press the space bar to complete or continue the list.
Step4
Enter privileged EXEC mode.
Step5
Notice the prompt that indicates that the switch mode was “>” and is now “#.”
Step6
Enter the help (?) command at the privileged EXEC mode prompt. Use help to determine the keyword command that manages the system clock.
Step7
Your console should be displaying a prompt of “--More--” as it waits for you to press a key before displaying more output. Enter q to terminate continuation of the output.
Step8
Enter the clock ? command. You should see the context-sensitive help. Your output should resemble the following example:
SwitchX#clock ?
  set  Set the time and date
Step9
Set the system clock to the current time and date. Remember to use context-sensitive help to guide you through the process.
Step10
At the switch# prompt, enter sh? You should see another example of the context-sensitive help. Your output should resemble the following example:
SwitchX#sh?
show
Step11
Press the Tab key. You should see the command-completion feature in action. When enough letters of a command or keyword have been entered, the Tab key will complete the word and place a space so that it is ready to receive any further input.
Step12
Enter the show clock command. Your output should reflect the changes that you made by using the clock set command in Step 9. Your output should be similar to the following example:
SwitchX#show clock
10:45:25.073 UTC Tue Jul 10 2007
Activity Verification
You have completed this task when you attain this result:
  • You have used the system help facility and the command-completion facility.
