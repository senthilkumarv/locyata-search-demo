Task 5: Verify OSPF Routing and Plaintext Authentication
In this task, you will verify the operation and configuration of the OSPF routing protocol and plaintext authentication. You will do this by using several show commands.

Complete these steps on the workgroup router:

Activity Verification
You have completed this task when you attain these results:
  • You have verified the operation and configuration of the OSPF routing protocol by using the appropriate show and debug commands.
