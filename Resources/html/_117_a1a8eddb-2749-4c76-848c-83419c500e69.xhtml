Task 1: Remove Any Residual Configuration from Your Router
In this task, you will start the workgroup router and remove the current configuration that has your router act as a PC host, returning the router to its default configuration.
Activity Procedure
Complete these steps:
Step1
Establish a connection to the lab equipment to access your workgroup router console.
Step2
If you are prompted for a router console password, try a password of cisco or sanjose (or consult the instructor for a password).
Step3
Enter privileged EXEC mode. If you are prompted for a privileged EXEC password, try a password of sanfran (or consult the instructor for a password).
Step4
Enter the command erase startup-config, to erase the startup configuration of your workgroup router.
Activity Verification
You have completed this task when you attain this result:
  • You have erased the startup configuration.
