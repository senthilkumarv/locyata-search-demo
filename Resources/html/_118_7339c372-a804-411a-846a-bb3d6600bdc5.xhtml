Task 2: Reload the Router and Observe the Startup Output
In this task, you will observe the output of the router. This should be similar to the output that was obtained when you observed your workgroup switch being reloaded.
Activity Procedure
Complete these steps:
Step1
Enter the command reload. Confirm the question to continue with reload by using the Enter key. Your output should resemble the following example:
PC_X#reload
Proceed with reload? [confirm]
.
Step2
