Lab Preparation
This lab activity has you make IP addressing changes to your workgroup. It also requires that your instructor make changes to the core configuration. Verify with your instructor that the core configuration has been updated.
In this lab activity preparation, you will configure your workgroup switch to participate in the default management VLAN.
Activity Procedure
Complete these steps on your workgroup switch:
Step1
From global configuration mode, enter interface configuration mode for the management VLAN with the command interface vlan 1.
Step2
Assign an IP address to the management VLAN interface of your workgroup switch. Use the IP address that is listed in the table in the Job Aids section of this lab.
Step3
Assign a default gateway to your workgroup switch. All workgroup switches will share the same default gateway. Use the address of the core router with the command ip default-gateway 10.1.1.3.
Activity Verification
You have completed this task when you attain these results:
  • You have configured your workgroup switch to participate in VLAN 1 with the proper IP address.
  • You have assigned your workgroup switch a default gateway of the core router.
