Task 3: Remove the ACLs from the Serial Interface
In this task, you will clean up after the lab so that the configuration changes that you made here do not negatively affect the next lab activity. It is important to complete this task.
Activity Procedure
Complete these steps on the workgroup router:
Step1
Enter interface configuration mode for your serial interface.
Step2
Remove all access groups from the serial interface.
Step3
Enter global configuration mode.
Step4
Remove both ACLs.
Step5
Save your running configuration to NVRAM.
Step6
Notify your instructor that you have completed the activity.
Activity Verification
You have completed this activity when you attain these results:
  • You have removed all access groups from the serial interface.
  • You have removed both ACLs in global configuration mode.
