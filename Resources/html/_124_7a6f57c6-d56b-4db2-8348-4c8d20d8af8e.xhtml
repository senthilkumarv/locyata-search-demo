Task 1: Enter the Initial Router Configuration
In this task, you will enter a basic router configuration.
Activity Procedure
Complete these steps:
Step1
From your workgroup router, access privileged EXEC mode.
Step2
From the privileged exec mode, enter global configuration mode.
Step3
Assign your workgroup router a hostname of RouterX, where “X” is your workgroup letter (A, B, C, D, E, F, G, or H).
Step4
Enable the first Ethernet interface (Fa0/0) of your workgroup router.
Step5
Enter the IP address of your assigned workgroup router. For this information, refer to the visual objective for this lab.
Step6
Provide a description for the FA0/0 interface describing to what device it connects.
Step7
Save your running configuration to the startup configuration.
Activity Verification
You have completed this task when you attain these results:
  • You have entered your workgroup router initial configuration.
