Task 2: Validate the Router Configuration
You will use show commands to check that the router configuration matches your requirements and that the configuration is saved to the startup configuration in the startup-config file.
Activity Procedure
Complete these steps:
Step1
Enter the command show running-config. Observe the output and validate that the passwords are set and match those that you entered in Task 1; also check that the interface Fa0/0 has the IP address that is assigned for your workgroup router and does not have the shutdown command applied to the interface. The following is an excerpt from the output; your display should be similar.

..Text omitted!
Step2
Enter the command show startup-config. Observe the output and validate that the information that you verified in Step 1 matches. This demonstrates that the setup command saved the configuration to both the running configuration and startup configuration.
Activity Verification
You have completed this task when you attain these results:
  • Your output of the show running-config command matched your input in Task 1.
  • Your startup configuration was the same as your running configuration.
